My first post

Blaine Mooers

April 19, 2025

This is my first attempt at using Hugo to develop a blog post. I’ve been using a different template that is actually quite complex and awkward to deal with. What I like about this approach is that I can generate the text inside of Emax in an orn file and then the changes that are made are automatically shipped out to the web page in HTML. This means that I can build out using the whisperer software to carry out dictations that make up at least the first draft of a blog post.
